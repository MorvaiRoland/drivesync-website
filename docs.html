Telepítési Útmutató

Üdvözöl a DriveSync hivatalos dokumentációjában. Itt minden információt megtalálsz a szoftver használatáról.

Tipp: A program futtatásához nem szükséges előre telepített Java, mivel a DriveSync tartalmazza a saját futtatókörnyezetét.

1. Letöltés és Kicsomagolás

Töltsd le a legfrissebb DriveSync.zip fájlt a főoldalról. A Windows beépített csomagolójával vagy WinRAR-ral csomagold ki egy tetszőleges mappába.

2. Indítás

Keresd meg a mappában a futtatható fájlt:

DriveSync.exe

Kattints rá duplán. Az első indításnál a Windows SmartScreen rákérdezhet a futtatásra, válaszd a "További információ" -> "Futtatás mindenképp" opciót.

Bejelentkezés

Használhatsz hagyományos email/jelszó párost, vagy a kényelmesebb Google Login gombot. Utóbbi esetben az adataid automatikusan szinkronizálódnak a felhőfiókoddal.
